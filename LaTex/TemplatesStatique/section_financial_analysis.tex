\subsection{Analyse de Sensibilité}

\highlightbox{
\textbf{Guide pour l'Analyse de Sensibilité:}
\begin{itemize}
\item \textbf{Scénario pessimiste:} Décrivez l'impact si les économies sont 20\% inférieures aux prévisions
\item \textbf{Scénario optimiste:} Décrivez l'impact si les économies sont 20\% supérieures aux prévisions
\item \textbf{Variables critiques:} Identifiez les facteurs qui ont le plus d'impact sur le ROI
\item \textbf{Points de bascule:} À quel niveau de performance le projet devient-il non rentable?
\end{itemize}

Incluez ici votre analyse spécifique montrant la robustesse financière du projet dans différents scénarios.
}

\subsection{Justification de l'Investissement}

\infobox{
\textbf{Points clés à inclure dans votre justification:}
\begin{enumerate}
\item \textbf{Retour rapide:} Expliquez quand les bénéfices commencent à se matérialiser
\item \textbf{Coût d'opportunité:} Quel est le coût de ne pas agir maintenant?
\item \textbf{Effet de levier:} Comment l'investissement génère-t-il de la valeur multiplicative?
\item \textbf{Flexibilité:} Quelles options futures cette solution ouvre-t-elle?
\item \textbf{Alignement stratégique:} Comment s'intègre-t-elle aux objectifs corporatifs?
\end{enumerate}
}
